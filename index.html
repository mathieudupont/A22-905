Procédure de débogage (ou « MON CODE NE FONCTIONNE PAS! »)
  • 1- Ne pas paniquer.
  • 2- Ouvrir la console.
  • 3- Lire les messages d'erreur dans la console : s'ils mentionnent une ligne spécifique problématique de votre code, aller voir cette ligne dans votre code.
  • 4- Vérifier l'écouteur : Mettre un console.log() dans la fonction appelée par l'écouteur et vérifier si ça s'affiche bien dans la console lorsque l'évènement écouté se produit.
  • 5- Appeler la fonction dans la console et vérifier si elle produit bien l'effet désiré.
  • 6- Vérifier si l'état d'une variable globale change : mettre des console.log() avant et après la modification de la variable.
